ZBrush は以下の4つの方法で参考にする画像を表示することができます。

1. Floor Grid 機能を使う

Floor Grid » ZBrush Docs http://docs.pixologic.com/reference-guide/draw/floor-grid/
Draw | Floor でグリッドに対して画像を貼り付けることができます。一般的な下絵機能の他にモデルに投影もできるらしいけど使ったことがない・・。いわゆる三面図みたいなのがある場合にはこれが良いと思います。



2. Image Plane 機能を使う

Image Plane » ZBrush Docs http://docs.pixologic.com/user-guide/zbrush-plugins/image-plane/
カメラの回転には対応していないものの、カメラのスケールや回転などが記録されるので、下絵にピッタリあった状態に簡単に復帰できるのがとても便利です。
Texture | Image Plane から設定できます。Store View で視点を保存するのを忘れないようにしてください。


3. See-through 機能を使う

See-through のスライダを上げるとウィンドウが半透明になるので、後ろに画像を置いて使います。シンプル。公式の簡単なチュートリアルではこれを使っていました。



4. Spotlight 機能を使う

Spotlight » ZBrush Docs http://docs.pixologic.com/user-guide/3d-modeling/painting-your-model/spotlight/
Texture | Add To Spotlight で追加できます。どちらかというと画像転写用なので、表示していると一部動作が異なるブラシがあったりするので、あんまりお勧めできないのですが、画像のサイズを調節したり複数の画像を使うことができるので、たまに使うことがあります。
Shift+Z で Spotlight 自体のオンオフを、Z でダイアルでの編集モードのオンオフを切り替えられます。


まとめ

私は Image Plane をベースにして、転写にも使っている関係で Spotlight をたまに使います。Floor Grid のも便利そうなので今度使ってみたいなって思っています。
